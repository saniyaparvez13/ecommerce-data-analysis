# Ecommerce Analytics Dataset
Project Overview

This repository contains a large-scale ecommerce transaction dataset used for data analysis and business insights. The dataset helps explore customer behavior, product performance, and revenue patterns using Python-based data analysis techniques.

It is suitable for data analytics practice, portfolio projects, and visualization exercises.

# Dataset Information
File Name: ecommerce_large.csv
Format: CSV (Comma-Separated Values)
Type: Ecommerce Data
Scope: Large synthetic/real-world style dataset  

# Features (Columns)
Column Name ->	Description
CustomerID ->	Unique identifier for each customer
Age	-> Age of the customer
Category	-> Product category purchased
PurchaseAmount ->	Amount spent per transaction

# Objectives

The main goals of this project are:
> Analyze overall revenue performance
> Understand customer spending behavior
> Segment customers by age groups
> Evaluate category-wise sales performance
> Identify top customers based on spending
> Visualize key business insights


# Key Analyses Performed
* Revenue Analysis
> Total Revenue
> Average Purchase Value
> Maximum & Minimum Purchase Amount
* Customer Insights
> Highest Spending Customer
Top 10 Customers by Revenue
* Demographic Analysis
> Age Group segmentation (18–25, 26–35, 36–45, 46+)
> Spending behavior across age groups
* Category Performance
> Category-wise total revenue
> Average purchase per category
> Unique customer count per category

# Visualizations

The following visualizations are used:
📊 Bar Chart → Category vs Revenue
📦 Box Plot → Purchase Amount Distribution
📉 Histogram → Purchase Amount Frequency
